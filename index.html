#01 scope of var, let, and const

var
  • Function scope.
  • Hoisted.
  • Allows redeclaration and reassignment.
let
  • Block scope.
  • Not hoisted.
  • Allows reassignment but not redeclaration
const
  • Function scope.
  • Must have a value when declared.
  • Cannot be reassigned (for primitives), but allows modifications for objects and arrays.

#02 use cases of null and undefined

null

Represents intentional absence or emptiness

undefined

Represents the absence of a value when one is expected

#03 What do you mean by REST API?

a REST API (Representational State Transfer Application Programming Interface) is a standardized way for software applications to communicate over the internet. It uses HTTP methods (like GET, POST, PUT, and DELETE) to access and manipulate resources (such as data) on a server. REST APIs are known for their simplicity, scalability, and use of URLs to identify resources. They are widely used for building web services and enabling interactions between different software systems.
